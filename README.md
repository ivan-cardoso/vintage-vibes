# Vintage Vibes
**Ecommerce de productos vintage**

## *Productos:*
 - **Camperas**
 - **Camisas**
 - **Camaras analógicas**
 - **Discos de vinílos**  

## *Tecnologías utilizadas:*
 - **`React Js`**
 - **`React Context`**
 - **`React Bootstrap`**
 - **`Scss`** 
 - **`Firebase`** 

**El objetivo** de este proyecto fue poner a prueba mis habilidades de Front-End.
